# 在线工具网站项目文档

## 项目概述

这是一个功能丰富的在线工具网站，提供多种实用工具，包括代码格式化、JSON处理、时间戳转换、UUID生成、图片处理、二维码生成、思维导图、日历工具、IP工具和反应测试等功能。网站采用现代化的前端技术栈，提供直观、易用的用户界面，支持响应式设计，可在各种设备上正常使用。

## 技术栈

### 前端

- **框架**：Vue 3
- **构建工具**：Vite
- **类型系统**：TypeScript
- **样式**：SCSS
- **状态管理**：Pinia
- **路由**：Vue Router
- **代码高亮**：Prism.js
- **图标**：Lucide Vue

## 安装和运行

### 前端

1. 安装依赖

```bash
npm install
```

1. 启动开发服务器

```bash
npm run dev
```

1. 构建生产版本

```bash
npm run build
```

## 使用说明

1. **访问网站**：打开浏览器，访问 `http://localhost:3000`（或开发服务器启动时显示的其他端口）
2. **选择工具**：在导航栏或首页卡片中选择需要的工具
3. **使用工具**：
   - 输入相关内容
   - 调整参数设置
   - 点击执行按钮
   - 查看结果
   - 复制或下载结果
4. **工具特性**：
   - 实时预览
   - 历史记录（部分工具）
   - 批量处理（部分工具）
   - 自定义设置
5. **全局功能**：
   - **夜间/白天模式**：点击顶部导航栏的月亮/太阳图标切换
   - **主题色**：点击顶部导航栏的调色板图标选择主题颜色
   - **语言切换**：点击顶部导航栏的地球图标选择语言（中文、英语、日语）

## 性能优化

- **代码分割**：使用Vue Router的动态导入实现路由懒加载
- **组件优化**：使用defineAsyncComponent实现组件懒加载
- **图片优化**：使用WebP格式和响应式图片
- **编译优化**：利用Vite的ES模块和按需编译
